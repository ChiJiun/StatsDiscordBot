State

H₀ (Null Hypothesis): There is no association between age group and whether or not a subscriber watches five or more genres per week.

H₁ (Alternative Hypothesis): There is an association between age group and whether or not a subscriber watches five or more genres per week.

Significance Level (α): 5%

Plan

Random: The survey stated that 8,866 subscribers were all randomly selected.

Independence: The sample size of 8,866 subscribers is less than 10% of the total population.

Expected counts:

E(18–34, Yes) = (2191 × 972) / 8866 ≈ 240.2

E(18–34, No) = (6675 × 972) / 8866 ≈ 731.8

E(35–54, Yes) = (2191 × 2911) / 8866 ≈ 719.38

E(35–54, No) = (6675 × 2911) / 8866 ≈ 2191.62

E(55 or older, Yes) = (2191 × 4983) / 8866 ≈ 1231.42

E(55 or older, No) = (6675 × 4983) / 8866 ≈ 3751.58


All expected counts are greater than 5.

Do
χ² =
(240 − 240.2)² / 240.2 +
(741 − 731.8)² / 731.8 +
(669 − 719.38)² / 719.38 +
(2242 − 2191.62)² / 2191.62 +
(1291 − 1231.42)² / 1231.42 +
(3692 − 3751.58)² / 3751.58

χ² ≈
(−0.2)² / 240.2 +
(9.2)² / 731.8 +
(−50.38)² / 719.38 +
(50.38)² / 2191.62 +
(59.58)² / 1231.42 +
(−59.58)² / 3751.58

χ² ≈
0.0002 + 0.1156 + 3.5279 + 1.158 + 2.883 + 0.1982 ≈ 8.8833

Degrees of freedom (df) = (3 − 1)(2 − 1) = 2

p-value ≈ 0.01

Conclude

Because our p-value is much smaller than α = 0.05, we reject the null hypothesis.
There is sufficient evidence to suggest that there is an association between age group and whether or not a subscriber watches five or more genres per week.
